此题与上题的相似之处在于:需要用到相同的结论:
s[i]= arr[0] + arr[1] +…+arr[i]
s[j]= arr[0] + arr[1] +…+arr[j]
假设i > j,则
s[i] - s[j] = arr[j+1] + arr[j+2] + … + arr[i]

复杂度分析

理论上, 时间复杂度为O(n).但实际实现时,时间复杂度为O(nlogn).因为map的find方法查找一个数的时间复杂度为O(logn)
空间复杂度为O(n)
